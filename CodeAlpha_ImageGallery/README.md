# 🍽️ Food Image Gallery

A modern, responsive **Food Image Gallery** built using **HTML, CSS, and JavaScript**.  
The project focuses on clean UI design, uniform image layout, smooth animations, and interactive user experience.

---

## 📌 Features

- Responsive image gallery using CSS Grid  
- Fixed image dimensions for visual consistency  
- Light blue themed background for a clean UI  
- Hover effects with smooth transitions  
- Lightbox modal view for image preview  
- Next / Previous navigation inside lightbox  
- Category-based image filtering (Breakfast, Lunch, Desserts)  
- Mobile-friendly and cross-browser compatible  

---

## 🛠️ Technologies Used

- **HTML5** – Page structure  
- **CSS3** – Styling, layout, transitions, responsiveness  
- **JavaScript (Vanilla)** – Filtering, lightbox, navigation

## ⚙️ How It Works

1. Images are arranged using a responsive CSS Grid layout.  
2. All images have a fixed height and use `object-fit: cover` to maintain uniform size.  
3. Category buttons filter images dynamically using JavaScript.  
4. Clicking an image opens it in a lightbox modal.  
5. Navigation buttons allow moving between images smoothly.  

---

## 🚀 How to Run the Project

1. Download or clone the project.  
2. Open `index.html` in any modern web browser.  
3. No additional setup or dependencies required.
